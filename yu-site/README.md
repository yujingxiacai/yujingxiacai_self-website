# yu-site（yujingxiacai个人网站）

## 个人网站简介
#### 1.背景：
        基于对自己的vue的知识整体的复习，我决定做一个个人网站，它是集技术博客，生活日常和周边故事的一个响应式网站。

        vue.js,
        animate.css,
        viewport,
        ajax,
        json,
        懒加载，
        动态分页，
        随机动态壁纸
